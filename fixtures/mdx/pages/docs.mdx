# Rex Documentation

Welcome to the **Rex** framework documentation.

## Features

- File-system based routing
- Server-side rendering with V8
- MDX support for content pages

## Getting Started

To create a new Rex project:

```bash
npx create-rex-app my-app
cd my-app
rex dev
```

Visit [the homepage](/) to learn more.
